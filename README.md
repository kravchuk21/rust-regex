# Regex Library in Rust

by video [Regex Tsoding Session](https://www.youtube.com/watch?v=MH56D5M9xSQ).

## Quick Start

Make sure you have `rustc` available in `$PATH`.

### Turnstile Example

```console
$ rustc turnstile.rs
$ ./turnstile
```

### Regex Example

```console
$ rustc regex.rs
$ ./regex
```
